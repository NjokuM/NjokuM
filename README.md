## 🚀 Featured Projects

### [PickAParlay](https://github.com/NjokuM/PickAParlay)
> Full-stack NBA prop betting analyzer powered by machine learning.

An 8-factor weighted ML model grades NBA player props across consistency, matchup, injury status, home/away splits, and more. Builds optimized parlay combinations and tracks prediction accuracy over time.

**Stack:** Python · FastAPI · Next.js · React · Tailwind CSS · SQLite · NBA API

---

### [SurvivorPoolApp](https://github.com/NjokuM/SurvivorPoolApp)
> Cross-platform football survivor pool manager with live data integration.

Players join custom pools across 5 major European leagues, make weekly picks, and compete to outlast the field. Features live fixtures via API-Football, async result processing, and a mobile-first React Native UI.

**Stack:** Python · FastAPI · PostgreSQL · React Native · Expo · Railway

---

### [SwitchCardGame](https://github.com/NjokuM/SwitchCardGame)
> Online multiplayer card game built with Godot 4 and a Firebase backend.

Full implementation of the Switch card game with online multiplayer (session codes), local pass-and-play, and CPU opponents. Supports Windows, macOS, Linux, and iOS export.

**Stack:** Godot 4.3 · GDScript · Firebase Realtime Database · Firebase Auth

---

### [UnsubscribeFromEmail](https://github.com/NjokuM/UnsubscribeFromEmail)
> Python + React tool to automate bulk email unsubscription.

**Stack:** Python · React · CSS
